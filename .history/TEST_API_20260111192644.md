## Test Signup
```bash
curl -X POST http://localhost:5000/api/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "firstName": "John",
    "lastName": "Doe",
    "email": "john@example.com",
    "password": "password123"
  }'
```

## Test Login
```bash
curl -X POST http://localhost:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john@example.com",
    "password": "password123"
  }'
```

## Expected Flow:
1. User fills signup form with firstName, lastName, email, password
2. Frontend sends POST to http://localhost:5000/api/auth/signup
3. Backend validates, hashes password, creates user, generates JWT token
4. Frontend receives token and user data
5. Token stored in localStorage
6. User redirected to /menu

## Debugging:
- Check browser console for frontend errors
- Check terminal for backend errors
- Verify .env file has MONGO_CONN and JWT_SECRET
- Ensure MongoDB is accessible at the connection URL
